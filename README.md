# 🎯 AI案件ハンター

Claude Code / Codexで対応できる案件だけを見極め、応募から実行準備まで支援するローカル案件管理ツール。

## このアプリでできること

- クラウドワークス / ランサーズなどの案件本文を **手動貼り付け** で登録
- 12項目の **自動スコアリング** と **総合判定 (S/A/B/C/D)**
- **地雷警告** と **応募候補理由** の自然言語表示
- **応募文生成プロンプト** (ブラウザ版ChatGPT/Claudeに貼って使う長文)
- **応募文テンプレ下書き** (アプリ内で自動生成)
- **Claude Code 用実行プロンプト** / **Codex 用実行プロンプト** の自動生成
- **応募前/納品前チェックリスト** の自動生成
- **見積もり補助** (推奨/最低受注ライン/時給警告)
- **ダッシュボード** (受注状況・月30万円までの差分)
- **ステータス管理** (未応募〜検収済みまで)

## このアプリではやらないこと

以下は **絶対に実装しません**。クラウドソーシング各サイトの規約遵守と、ユーザー自身の安全のためです。

- 自動応募・自動ログイン・応募ボタン自動クリック
- 自動契約・自動納品
- 有料API (OpenAI API / Claude API 等) の利用
- AI禁止案件への応募支援
- 機密情報のコード直書きを前提とする設計

最終判断と実行は **必ず人間** が行う設計です。

## インストール / 起動

```bash
pip install -r requirements.txt
streamlit run app.py
```

ブラウザが自動で開き `http://localhost:8501` でアプリが表示されます。

初回起動時に `data/database.sqlite` が自動生成されます。

## 使い方 (操作フロー)

1. **案件登録** — クラウドワークス等で見つけた案件本文を貼り付けて登録
2. **スコア確認** — 案件詳細でスコア内訳と判定を確認
3. **応募前チェック** — 応募前チェックリストで地雷を確認
4. **応募文プロンプトをコピー** — ブラウザ版ChatGPT / Claudeに貼る
5. **ブラウザ版AIで応募文作成** — テンプレ感を消す
6. **自分で応募** — 各サイトの自分のアカウントから手動応募
7. **受注したら Claude Code / Codex 実行プロンプトをコピー** — 実装開始
8. **実装** — 仮定を提示させながら進める
9. **納品前チェック** — チェックリストで確認
10. **ステータス更新** — 受注/納品済み/検収済み等に反映

## 案件登録のコツ

- **案件本文は省略せず貼り付け** ること。スコアの精度が大きく上がります
- **予算は数字を入れる** こと(例: `30,000円〜50,000円`, `5万円`)
- 不明な箇所は無理に埋めず空欄でOK。後から再スコアリング可能です

## 応募文プロンプトの使い方

1. 「応募支援」ページで対象案件を選択
2. 「② ブラウザ版AIに貼る『応募文生成プロンプト』」の枠内をクリック
3. Cmd/Ctrl + A → Cmd/Ctrl + C で全選択コピー
4. ブラウザ版 ChatGPT または Claude のチャットに貼り付け
5. 出てきた応募文を確認・微修正してクラウドソーシング側に貼り付け

> ⚠ 「AIで全部やります」と書かないこと。
> 「必要に応じてAIツールも補助的に活用し、最終的な品質確認は私が行います」程度の自然な表現が推奨です。

## Claude Code / Codex 実行プロンプトの使い方

1. 「実行プロンプト」ページで対象案件を選択
2. タブで Claude Code 用 / Codex 用を切り替え
3. 枠内をクリック → 全選択コピー
4. Claude Code / Codex のセッションに貼り付けて実装開始
5. まず要件整理と仮定一覧が提示される → 確認して「そのまま進めて」と指示

## 注意点 (セキュリティ・規約)

- 機密情報 (Webhook URL / APIキー / アカウント情報) は **コードに直書きしない**。`.env` を使う設計を Claude Code / Codex に必ず指示してください
- スクレイピング案件では **robots.txt と利用規約を必ず確認**。自動ログインや個人情報取得は行いません
- 法務 / 医療 / 投資助言 など責任の重い案件は **D判定** で表示されます
- このアプリ自身も、クラウドソーシングサイトに対する自動アクセスや自動ログインは一切行いません

    ├── constants.py        # 定数 (プラットフォーム/カテゴリ/キーワード等)

## 思想 (この章は重要)

このアプリの本質は「AIに全部任せて危ない応募をする」ことではありません。

- Claude Code / Codex で **納品可能な案件だけを見極める**
- 応募前の **判断精度を上げる**
- 応募文作成を **速くする**
- 受注後の実行プロンプトを **高品質化する**
- 副業案件の **回転率を上げる**

副業で月30万円を継続的に達成するためのワークフロー支援ツールです。
